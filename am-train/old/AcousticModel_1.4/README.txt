---------  Projeto FalaBrasil -----------

-- Arquivos para criação do Modelo Acústico(MA):

Scripts:

Inicial.sh  - Prepara os dados para treino
FrontEnd.sh - Extrai os parâmetros
AM_01.sh    - Estima modelos monofones
AM_02.sh    - Cria modelos trifones
AM_03.sh    - Realiza Clustering de trifones por Árvore de decisao fonética
AM_04.sh    - Incrementa o número de gaussianas
Test.sh     - Exemplos do uso de vários decodificadores

diretórios:

confs/      - Contém arquivos de configuração
scripts/    - Contém outros scripts (JAVA, perl e bash) que auxiliam na Criação do MA.
util/       - arquivos utilizados na criação do MA.
