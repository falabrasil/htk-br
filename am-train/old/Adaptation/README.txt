# Contém os scripts e arquivos necessários para realizar a adaptação de locutor

### Scripts
- Init.sh 	- Realiza os passos iniciais para adaptação de locutor (criar o arquivo MLF)
- makeMLLR.sh 	- Realiza a adaptação do tipo MLLR
- MakeMap.sh 	- Realiza a adaptação MAP
- MakeALL.sh 	- Executa todos os scripts

### Arquivos e pastas
- mfc_train.list - lista dos arquivos mfccs que serão utilizados para adatação
- scripts/	- diversos scripts utilizado no processo de adaptapação
- speakers/	- contem arquivos de vários locutores para adaptação e scripts para gravação
- util/		- arquivos de configuração, dicionário, etc 
- others/	- arquivos MLF, list originais do spoltech
